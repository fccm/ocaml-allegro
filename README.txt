
  This is an OCaml binding for Allegro.
  Allegro is a cross-platform library intended for use in computer games
  and other types of multimedia programming.
  OCaml is a fast and powerful language for application development.

  Get Allegro from http://alleg.sourceforge.net/
  Get OCaml from http://caml.inria.fr/

  This binding is released under the GPL licence,
  see the LICENCE_GPL.txt file that comes with this package.
  There is NO WARRANTY!

  To build and install the binding, run:
    make
    su -c 'make install'
  to test the bindings with a "hello world" example, run:
    make test
  and for the module documentation:
    make doc

  The module documentation will give you the list of the binded Allegro
  functions with their ocaml type. For a description of each function
  please refer to the Allegro documentation:

    http://alleg.sourceforge.net/stabledocs/en/allegro.html
